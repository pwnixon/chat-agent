# Chat Agent

Archera Chat Agent UI prototype — an AI-powered cloud cost optimization assistant.

## Setup

```bash
git clone https://github.com/pwnixon/chat-agent.git
cd chat-agent
./setup.sh
```

Then open `http://localhost:5174`.

## Demos

| URL | Description |
|-----|-------------|
| `/` | Demo index |
| `/?demo=welcome` | Welcome / empty state |
| `/?demo=chat` | Savings analysis conversation |
| `/?demo=dev` | Dev mode (fast responses) |
